This is an website HTML/Javascript emulation of the Weather Channel's Local on the 8's IntelliSTAR system.
Relies on Node.JS to run self-hosted.
For the voice narration, requires access to a PiperTTS server to convert the voice data to spoken words.

As of January 2026 the project was internal and unversioned. Now that it is mostly complete will be
updating the github repo and will be including a version on the main UI screen.
